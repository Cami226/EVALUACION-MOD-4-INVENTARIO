# EVALUACION-MOD-4-INVENTARIO

INVENTARIO

Descripción del reto
Desarrollar una aplicación de consola en Java que permita gestionar un inventario de productos para una pequeña tienda. Los usuarios podrán:

Agregar productos al inventario.

Buscar productos por nombre.

Listar todos los productos registrados.

Eliminar productos por ID.

Realizar operaciones básicas como cálculos de stock total y precios.

La aplicación deberá seguir principios básicos de diseño orientado a objetos (uso de clases, encapsulamiento, etc.), buenas prácticas del lenguaje Java (nombres significativos, convenciones de estilo, separación por responsabilidades) y además incluir una suite de pruebas unitarias con JUnit para asegurar su correcto funcionamiento.
